Installation


```npm install level```

You can perform a modular installation using the attribute '<a href="http://kimera.comli.com">km-include</a>' alongside javascript in the following manner:

 ``` 
<pre>
	<body km-include="components acordeon">
		<!-- Contenido de tu web -->
	</body>
</pre>
```
